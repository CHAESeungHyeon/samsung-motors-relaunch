# Samsung Motors Redesign

## 📌 프로젝트 소개

기존 자동차 브랜드 사이트를 분석하여 사용자 경험을 개선하고,
브랜드 아이덴티티를 강화하는 방향으로 리브랜딩 기획 프로젝트입니다.

## 📂 주요 기능

* 메인 비주얼 배너 슬라이드 구현
* 반응형 레이아웃 구성
* 내비게이션 인터랙션 설계

---

## 🧠 구현 포인트

* 사용자 시선 흐름을 고려한 레이아웃 구조 설계
* 브랜드 감성을 강조한 비주얼 구성
* 다양한 디바이스 환경 대응
